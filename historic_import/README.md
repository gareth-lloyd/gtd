# historic_import

Frozen export of the user's GTD data from Nirvana, kept for historical
reference only. NOT loaded by the running app — see
`gtd_core/importer_nirvana.py` for the importer that consumed this once.

If the contents are no longer needed for retrospective lookup, this
directory can be deleted along with `gtd_core/importer_nirvana.py` and
its management command.
